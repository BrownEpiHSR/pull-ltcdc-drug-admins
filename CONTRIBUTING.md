# Contributing to pull-ltcdc-drug-admins

## How Can I Contribute?

### Suggesting Ideas or Reporting Errors (via Issues)
If you find an error or have an idea for a new addition but do not plan to write the code yourself:
* Check the **Issues** tab to see if it has already been reported.
* If not, open a new issue.
    *   If you are suggesting a new addition to the repo, use the *Suggest Addition* template.
    *   If you are reporting a code or documentation issue, use the *Code or Documentation Issue* template.

### Directly Implementing Changes (via Pull Requests)
If you would like to directly update the code or documentation:
1. **Fork** the repository.
2. Create a new branch for your feature or fix.
3. Submit a **Pull Request**. 

## Peer Review Process
All Pull Requests will be reviewed by the core research team.
